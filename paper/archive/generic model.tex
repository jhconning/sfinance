\subsection{Generic model}

Why full returns of a project cannot be pledged to investors.

$Z_{1}$ are project returns

$Z_{0}$ are pledgeable project returns

$I$ is project size.

$Z_{0}\leq I\leq Z_{1}$

The project is not self financing: $I-Z_{0}>0$ investment will require a
positive contribution from the entrepreneur (collateral).

Entrepreneurial rent $Z_{1}-Z_{0}$

Attempts to reduce entrepreneur's share below $Z_{1}-Z_{0}$ will hurt
investors (by changing incentives).

\bigskip 

Project can go forward iff pledgeable income exceeds the project's net
financing need

$Z_{0}+A\geq I$

So entrepreneur must have $A$\TEXTsymbol{>}\underline{$A$}$=I-Z_{0}$

Anybody with less will be credit rationed. \ 

For credit rationing need firms to be \emph{capital poor }, which is to say $%
A$ is less than entrepreneurial rent $Z_{1}-Z_{0}$

\bigskip Another way to say this is that projects only go ahead if NPV of
project $Z_{1}-I$ exceeds entrepreneurial retn

\bigskip $Z_{1}-I\geq Z_{1}-Z_{0}-A$

In HT they consider equity participation by the entrepreneur $I-A$
